Paste Grade Export

Paste rows directly from your grade system. Each row should have a student ID, name, section, grade level, and four quarter grades like C(73.28). The system average column at the end is ignored.

Which grade to use? Per Carroll County grading policy, award the higher of the two methods — the highlighted box is the one to report.
Quality Points: Each quarter letter converts to a point value (A=4, B=3, C=2, D=1, F=0). Those four values are averaged, then converted back to a letter.
Percentage Average: The four raw percentages are averaged directly, then converted using the 10-point scale (rounds up from .5).
A (90–100)
B (80–89)
C (70–79)
D (60–69)
F (below 60)
